# GetPathway

Returns a specific pathway.

## JavaScript library method

```javascript
patientportal.pathways.getPathway({
    pathway: <pathway>
});
```

## HTTP Method

GET

## ****Url****

/patientportalapi/pathways/pathway

## URL Parameters

| pathway | int | Pathway ID |
| --- | --- | --- |

## Returns

PathwayData
