# GetDocument

Returns a document details.

## JavaScript library method

```javascript
patientportal.documents.getDocument({
    document: <document>
});
```

## HTTP Method

| Verb | URL                                               |
|:-----|:--------------------------------------------------|
| GET | `/patientportalapi/documents/document` |

## URL Parameters

| document | document | The key of the document provided by the API. |
| --- | --- | --- |

## Returns

DocumentData
