# GetUserRights

Returns the list of possible user rights.

## JavaScript library method

```javascript
patientportal.users.getUserRights();
```

## HTTP Method

| Verb | URL                                               |
|:-----|:--------------------------------------------------|
| GET | `/patientportalapi/users/user-rights` |
