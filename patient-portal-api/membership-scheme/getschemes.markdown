# GetSchemes

Returns list of possible membership schemes the patient can join.

## JavaScript library method

```javascript
patientportal.membershipScheme.getSchemes();
```

## HTTP Method

GET

## ****Url****

/patientportalapi/membership-scheme/schemes

## Returns

MembershipSchemeData\[\]
