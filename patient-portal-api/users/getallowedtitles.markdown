# GetAllowedTitles

Gets allowed titles (for example: Mr, Mrs, etc.).

## JavaScript library method

```javascript
patientportal.users.getAllowedTitles();
```

## HTTP Method

| Verb | URL                                               |
|:-----|:--------------------------------------------------|
| GET | `/patientportalapi/patient/allowed-titles` |

## Returns

string[]
