# GetCase

Returns a case.

## JavaScript library method

```javascript
patientportal.cases.getCase({
    case: <case>
});
```

## HTTP Method

GET

## ****Url****

/patientportalapi/cases/case

## URL Parameters

| case | string | The key of the case. |
| --- | --- | --- |

[CaseData](#_CaseData)
